Open TriviaQuiz

quiz application

The Open Trivia Quiz is a trivia quiz app that combines the power of React.js and the Open Trivia API to deliver an engaging learning experience. The app utilizes React's component structure to create a seamless user interface while the Open Trivia API provides a diverse range of questions on a wide variety of topics. The random selection of questions ensures that users are constantly challenged and learning something new. This project was a great opportunity for me to showcase my skills in developing interactive and fun user interfaces using React.js, and integrating third-party APIs to enhance the user experience.
